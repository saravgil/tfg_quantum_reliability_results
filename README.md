# Resultados numéricos y archivos de generación de figuras

Este repositorio contiene los resultados numéricos y los archivos de generación de figuras utilizados en el Trabajo Fin de Grado.

El objetivo del repositorio es facilitar la reproducibilidad de los resultados gráficos. Los datos numéricos ya están incluidos, por lo que las figuras pueden obtenerse simplemente ejecutando los archivos de plot correspondientes.

## Contenido del repositorio

* Archivos con resultados numéricos.
* Archivos de plot asociados a los distintos experimentos.
* Archivos necesarios para reproducir las figuras.

## Cómo generar las figuras

Para reproducir las figuras, descarga o clona este repositorio y ejecuta el archivo de plot correspondiente a cada experimento.

Por ejemplo:

```bash id="wstbph"
python exp1_plot.py
```

Cada archivo de plot lee los resultados numéricos correspondientes y genera las figuras asociadas.

## Nota

Se recomienda mantener la estructura de carpetas original para que los archivos de plot puedan localizar correctamente los datos numéricos.
